原题

题面请查看Codeforces 271B Prime Matrix。

也可在洛谷上查看：传送门。

题解

埃氏筛

思路

  1. 用埃氏筛求出$1$~$10^{5}+3$以内的所有素数；

  2. 求出矩阵内每个元素变为素数的操作次数；

  3. 为最后统计方便，记a[i][0]为第$i$行的操作总数，a[0][j]为第$j$列的操作总数；

  4. 统计答案。

细节

$10^5$经过操作后的素数是$100003$，所以埃氏筛要筛到$100003$。
