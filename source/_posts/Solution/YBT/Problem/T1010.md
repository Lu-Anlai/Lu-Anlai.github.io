### T1010 : 计算分数的浮点数值

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

两个整数$a$和$b$分别作为分子和分母，既分数$\frac{a}{b}$，求它的浮点数值(双精度浮点数，保留小数点后$9$位)。

#### 【输入】

输入仅一行，包括两个整数$a$和$b$。

#### 【输出】

输出也仅一行，分数$\frac{a}{b}$的浮点数值（双精度浮点数，保留小数点后$9$位）。

#### 【输入样例】

```
5 7
```

#### 【输出样例】

```
0.714285714
```

#### 【答案&代码】

```cpp
#include<cstdio>

double a,b;

int main(void){
    scanf("%lf%lf",&a,&b);
    printf("%.9f\n",a/b);
    return 0;
}
```
