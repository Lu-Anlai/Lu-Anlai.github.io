### T1011 : 甲流疫情死亡率

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

甲流并不可怕，在中国，它的死亡率并不是很高。请根据截止$2009$年$12$月$22$日各省报告的甲流确诊数和死亡数，计算甲流在各省的死亡率。

#### 【输入】

输入仅一行，有两个整数，第一个为确诊数，第二个为死亡数。

#### 【输出】

输出仅一行，甲流死亡率，以百分数形式输出，精确到小数点后$3$位。

#### 【输入样例】

```
10433 60
```

#### 【输出样例】

```
0.575%
```

#### 【答案&代码】

```cpp
#include<cstdio>

double a,b;

int main(void){
    scanf("%lf%lf",&a,&b);
    printf("%.3f%%\n",b/a*100);
    return 0;
}
```
