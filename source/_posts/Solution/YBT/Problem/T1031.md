### T1031 : 反向输出一个三位数

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

将一个三位数反向输出，例如输入$358$，反向输出$853$。

#### 【输入】

一个三位数$n$。

#### 【输出】

反向输出$n$。

#### 【输入样例】

```
100
```

#### 【输出样例】

```
001
```

#### 【答案&代码】

```cpp
#include<cstdio>

int num;

int main(void){
    scanf("%d",&num);
    printf("%d%d%d\n",num%10,(num/10)%10,num/100);
    return 0;
}
```
