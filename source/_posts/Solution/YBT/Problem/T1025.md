### T1025 : 保留12位小数的浮点数

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

读入一个双精度浮点数，保留$12$位小数，输出这个浮点数。

#### 【输入】

只有一行，一个双精度浮点数。

#### 【输出】

也只有一行，保留$12$位小数的浮点数。

#### 【输入样例】

```
3.1415926535798932
```

#### 【输出样例】

```
3.141592653580
```

#### 【答案&代码】

```cpp
#include<cstdio>

double num;

int main(void){
    scanf("%lf",&num);
    printf("%.12f\n",num);
    return 0;
}
```
