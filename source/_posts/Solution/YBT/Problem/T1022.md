### T1022 : 整型与布尔型的转换

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

将一个整型变量的值赋给一个布尔型变量，再将这个布尔型变量的值赋给一个整型变量，得到的值是多少？

#### 【输入】

一个整型范围内的整数，即初始时整型变量的值。

#### 【输出】

一个整数，经过上述过程后得到的结果。

#### 【输入样例】

```
3
```

#### 【输出样例】

```
1
```

#### 【答案&代码】

```cpp
#include<cstdio>

int n;

int main(void){
    scanf("%d",&n);
    printf("%d\n",n!=0);
    return 0;
}
```
