### T1051 : 分段函数

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

编写程序，计算下列分段函数$y=f(x)$的值。结果保留到小数点后三位。

```
y=−x+2.5;          (0  ≤x<  5)
y=2−1.5(x−3)(x−3)  (5  ≤x< 10)
y=x/2−1.5           (10 ≤x< 20)
```

#### 【输入】

一个浮点数$N(0 \leq N<20)$。

#### 【输出】

输出N对应的分段函数值：$f(N)$。结果保留到小数点后三位。

#### 【输入样例】

```
1.0
```

#### 【输出样例】

```
1.500
```

#### 【答案&代码】

```cpp
#include<cstdio>

double x,y;

int main(void){
    scanf("%lf",&x);
    if(0<=x&&x<5)
        y=-x+2.5;
    else if(5<=x&&x<10)
        y=2-1.5*(x-3)*(x-3);
    else if(10<=x&&x<20)
        y=x/2-1.5;
    printf("%.3f\n",y);
    return 0;
}
```
