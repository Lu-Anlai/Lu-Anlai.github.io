### T1020 : 打印ASCII码

> 时间限制: $1000 \text{ms}$ 内存限制: $65536 \text{KB}$

<!-- more -->

#### 【题目描述】

输入一个除空格以外的可见字符（保证在函数`scanf`中可使用格式说明符`%c`读入），输出其`ASCII`码。

#### 【输入】

一个除空格以外的可见字符。

#### 【输出】

一个十进制整数，即该字符的`ASCII`码。

#### 【输入样例】

```
A
```

#### 【输出样例】

```
65
```

#### 【答案&代码】

```cpp
#include<cstdio>

char ch;

int main(void){
    scanf("%c",&ch);
    printf("%d\n",ch);
    return 0;
}
```
